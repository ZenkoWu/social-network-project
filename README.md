# Social-network
Проект представляет собой небольшую соц. сеть. Пока приложение находится в разработке, но, возможно, вам будет интересен его функционал, например появление 
бургерного меню при уменшении размера страницы :)

Стек: 
- Javascript 
- React 
-Redux
- HTML
- Bootstrap
- Css
-TypeScript
-UI material
